# tic-tac-toe

My minimalist version of tic-tac-toe.

---

## Install

Download latest version from [releases](https://github.com/TheBlackbird14/tic-tac-toe/releases/tag/V1.0).

### Unzip

```bash

unzip cgl.{version}.zip

```

### Compile

Requires make to be installed

```bash

make cgl

```

### Execute

```bash

./cgl

```
